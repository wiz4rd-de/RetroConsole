Some systems need BIOS files that cannot be distributed with RetroConsole.

PlayStation 1: copy your BIOS dumps (scph5500.bin, scph5501.bin, scph5502.bin)
into this directory. You can drop them onto the "ROMs" network share into any
folder and move them here, or use a USB stick.

Game Boy Advance games run without a BIOS, but gba_bios.bin placed here
improves compatibility.
