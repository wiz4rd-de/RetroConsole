# RetroConsole

A Retrogaming Distro to be directly installed onto bare metal.

## Uses

- Retroarch (emulator)
- ES-DE (frontend)
- Based on a small linux distro

## Installation

0. Flash ISO to bootable USB-Stick or Flash-Drive.
0. Plug into target computer
0. Turn on

This starts a mainly unattended installer.

**Careful!** This wipes the internal drive

## Play

After installation has finished, the computer boots up directly into ES-DE.
